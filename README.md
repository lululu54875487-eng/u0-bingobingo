# 小u0賓果bingo

可愛柔和小雞黃色風格的多人即時賓果小遊戲。

## 功能

- 建立或加入房間，房間連結可分享。
- 參賽者上限 6 人，額滿後仍可當觀眾聊天互動。
- 房主可選題庫：混合、數字、水果、動物、自訂。
- 房主可選決勝條件：1 條線、2 條線、3 條線。
- 開始後系統自動隨機叫號，玩家賓果卡自動蓋章。
- 即時計算每位玩家線數，先達標者獲勝。
- 旁邊留言與表情符號互動。

## 執行

```sh
npm install
npm start
```

開啟：

```text
http://localhost:3000
```

若 3000 連接埠已被使用，可改用：

```sh
$env:PORT=3001; npm start
```
